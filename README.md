# OCTAVIRE

Landing page for OCTAVIRE — the first end-to-end design software built for professional tattoo artists.

Single-file static site: `index.html`.

## Deploy

This is plain static HTML/CSS/JS — any static host works.

- [Vercel](https://vercel.com) — connect this repo and it auto-deploys on push (zero config)
- [Netlify](https://www.netlify.com) — same, zero config
- GitHub Pages — enable Pages in repo settings, choose `main` branch

## Stack

- Vanilla HTML / CSS / JS
- Three.js (loaded via CDN importmap) for the GhostCursor shader
- Google Fonts (Inter) for typography
- No build step
